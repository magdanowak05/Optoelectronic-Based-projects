# Link Budget Calculator
Program do obliczania link budgetu i innych parametrów z nim związanych, na podstawie wybranych parametrów. Na Laser Communication Systems (LCS).

# PhotonW
A system programmed to warn of exceeding the permissible intensity of UV light can be used in many different forms - from human needs to research instruments of space probes.

# Light Coded
A system that detects light signals, determines Morse code characters based on their duration and the order in which they appear, and then decrypts messages. Arduino-based
